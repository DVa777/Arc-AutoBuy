# Arc AutoBuy

### Tired of buying the same items and setting up your loadout every time you launch ARC Raiders?

Arc AutoBuy automates the entire process by purchasing your selected items from traders and instantly equipping them to your loadout through a simple GUI.

> ⚠️ **Project Status:** This project is still under active development. Some features may not work as expected.

> 💡 This project is completely free and created to simplify gameplay and help players reduce repetitive in-game actions.

---

## ✨ Features

- 🛒 Automatically buys items from traders.
- 🎒 Automatically equips purchased items to your loadout.
- 🖥️ Simple and easy-to-use graphical interface.
- ⚡ Saves time by automating repetitive actions.

---

## 📥 Downloads

There are **two versions** of Arc AutoBuy available in the **Releases** section.

### 🖥️ GUI Version
- Graphical interface only.
- No console window.
- Recommended for everyday use.

### 🐞 Debug Version
- Includes both the graphical interface **and** a debug console.
- Displays logs and error messages in real time.
- Required for reporting bugs.

> **No Python installation is required** to run either version. Simply download the `.exe` file from the Releases page.

---

## 🛠️ Building from Source

The project was developed and compiled using **Python 3.9** and **PyInstaller**.

```bash
git clone https://github.com/yourusername/ArcAutoBuy.git
cd ArcAutoBuy
pip install -r requirements.txt
python "Arc AutoBuy UI.py""
```

---

## 🐛 Reporting Bugs

If you encounter an issue, please use the **Debug Version** and include the console output when creating a GitHub issue.

Please include:

- A description of the problem.
- Steps to reproduce it.
- A screenshot (if applicable).
- **The complete output from the Debug Console.**

Bug reports without the debug console output may be difficult to investigate.
